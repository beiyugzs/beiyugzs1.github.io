Hexo+github搭建博客、无需服务器

准备工作，下载需要的软件和github账号

1.账号注册/账号最好输入英文，因为中文的话，下面搭建好之后，网址可能无法访问
2. Git
3. node.js

下载好之后

  • 在桌面右击选择” Git bash Here “测试一下有没有安装好文件，按顺序依次输入指令回车
  1. node -v
  2. npm -v
  3. git –version
  4. npm install hexo-cli -g

搭建方法

  • 在任意磁盘创建一个文件夹命名为 blog 或者其他名字也可以，推荐使用blog
  • 在blog文件夹里面右击打开Git bash Here

  1. 之后按顺序输入指令回车

    初始化hexo

    1. hexo init
      安装依赖文件
    2. npm install
    3. hexo s
    • 输入第三个指令之后会出现下图中的” localhost:4000 “
    • 这时候给了一个本地访问地址，目前只有你可以访问，想要其他人也能访问
      就需要上传到github，进行一个远端访问
    • 先Ctrl+c停止本地访问，才能进行下一步
      创建Github资料库

  • 如图 点击”New repository “

  • 创建资料库

  • 按下面图中提示进行输入， 之后点击 “ cteate repository ” 创建

  • 不然无法访问

  • 创建好之后如下图,不要关闭页面划红线的链接

  • 后面还需要用到

安装deployer插件

  • 输入指令

    npm install hexo-deployer-git –save

  • 我们用编辑器打开下图文件

推送至Guhub
  • 输入以下提示指令，更新GitHub登录账号

    git config –global user.eamil “你注册GitHub用的邮箱”

git config –global user.name “你注册GitHub用的名称”

  • 输入下面指令进行推送

hexo g -d

  • 然后会弹出GitHub登录界面，根据提示填写，填写完之后点Login登录，如下图
  • 登陆之后，会自动进行推送
  • 推送成功之后会出现下图红线画出的命令,则表示推送成功
  • 之后就可以使用下图填写内容进行登录
